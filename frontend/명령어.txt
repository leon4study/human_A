# node.js 다운
- node.js가 설치되어 있지 않은 경우 먼저 다운로드

# frontend로 이동 후 명령어 입력
> cd frontend

# 패키지 설치
> npm install					# 패키지 새로 설치
> npm run build					# 에러 없이 빌드 되는지 검사
> npm run dev					# 개발 서버 실행

# 패키지 설치에 실패할 시 삭제 후 재설치
> Remove-Item -Recurse -Force node_modules		# 기존 설치 패키지 전부 삭제
> Remove-Item -Force package-lock.json		# 기존 설치 기록 삭제
> npm install
> npm run build
> npm run dev
